# Проверка ссылок в Яндекс.Директе

Скрипт обходит активные кампании в кабинете Яндекс.Директа, извлекает ссылки из объявлений и сообщает обо всех ответах, отличных от 500 (по условию это считается рабочим ответом).

## Подготовка
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Задайте учетные данные через переменные окружения или аргументы командной строки:
   - `YANDEX_API_TOKEN` – OAuth токен.
   - `YANDEX_CLIENT_LOGIN` – логин кабинета без `@yandex.ru`.

## Запуск
```bash
python check_links.py --token "$YANDEX_API_TOKEN" --client-login "$YANDEX_CLIENT_LOGIN"
```
Дополнительно можно указать `--timeout` (секунды ожидания ответа при проверке ссылок).

## Что делает скрипт
- Запрашивает все включенные кампании (`State = ON`, `Status = ACCEPTED`).
- Получает объявления в каждой кампании и извлекает ссылки из текстовых форматов.
- Проверяет каждую ссылку HTTP-запросом. Ответ 500 считается корректным; любой другой статус или ошибка запроса выводится как проблема.

Результаты выводятся в STDOUT, что позволяет перенаправить их в файл или систему мониторинга по необходимости.
